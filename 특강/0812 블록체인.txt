html 파일은 html로 시작해야한다

<> 엮인 것 -> 태그
태그들은 마지막에 </>로 마무리해야한다
<> - 여는 태그 / </> - 닫는 태그

<head> 태그 안에 브라우저에게 줄 정보들이 적혀있다.

<meta> 기본정보를 넘긴다
charset -> 캐릭터셋의 약자로 언어를 정하는 것 - 프로그램에서 캐릭터=문자를 뜻함
UTF-8 기본

http-equiv 호환성과 관련된 부분

name -> 설정할 이름 

title -> 브라우저에 뜨는 사이트 이름

여는 태그와 닫는 태그를 합쳐 엘리멘트라고 함
ex) <head> ~ </head>
열면서 바로 닫는 친구들도 있음(무조건적으로 자식이 없는 경우)

<body> -> 웹 페이지의 내용들
