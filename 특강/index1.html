굵은 글씨 강조, 긴급한 내용 같은 곳에 쓰임 ex) 알람 채팅을 쳤는데, 공지를 올림 : 긴급사항, 알림을 strong 으로 강조 그냥 굵은 글씨, 나중에 css에서 알 수 있다. ex) 삭제한 내용, 게시판에서 글을 삭제 그 글을 관리자는 확인가능해야함. insert 입력한 내용, 늘어난 내용 ex) 게시판에 글을 작성함, 신규 작성글 등으로 활용하여 쓸 수 있음 어떤 내용을 수정했을 때 2의 제곱 22 영어English 22 강아지동물 둘 다 기울었음, 이름이나 제목 같은 거, 가져온 말들(인용) ex) 명언 둘 다 기울었음, 강조 like strong

style : 속성, 그 태그의 크기, 외곽선, 위치, 배경, 등을 설정한다. 기초적인 방법 웹페이지가 처음은 html 파일로 주고받았음. 점점 발전하며 html파일이 너무 커져서 보기가 어려워짐 그래서 나온 것이 CSS 파일, 즉 Casecading Style Sheet로 HTML 파일 외부로 빼서 불러올 수 있게 함(보기도 편해짐=양이 줄어서) 이 방식은 나중에 배움

너비나 높이나 숫자를 이용한 수치들에 대한 단위 : px, %, em, rem, vm, vh

px : 픽셀, 한칸 한칸

% : 부모를 기준으로 한 %, 100% = 부모의 너비

em : 글씨 크기를 기준으로 한다. 부모의 글씨 크기

rem : 글씨 크기를 기준으로 한다. html의 글씨 크기, 모바일, PC한번에만들 때 많이 사용한다.

ex) 글씨 크기가 16px, 그럼 1em, 1rem은 16px 글씨 크기가 32px, 1em = 1rem = 32px, -> 0.5rem = 16px

vw,vh : viewport, 브라우저나 스마트폰의 크기를 기준으로 %를 따짐 100vw = 화면의 너비, 100vh 화면의 높이

width : 영역의 너비 max-width : 영역의 최대 너비 min-width : 영역의 최소 너비 %, vw, vh은 기준이 있고 그 기준의 비율로 너비를 설정 어떤내용을 넣었는데, 최소 100px은 있어야 제대로 뜬다. -> min-width 어떤내용을 넣었는데, 너무 크면 안된다. 1000px 이상 넘어가면 안된다. ->max-width
height : 높이 max-height : 영역의 최대 높이 min-height : 영역의 최소 높이 이미지 조절할 때 사용 이미지 조절할 때, 이미지에 대한 style을 할 때 다시 설명해주실 예정
