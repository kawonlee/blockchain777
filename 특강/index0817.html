width

너비나 높이, 숫자를 이용한 수치들에 대한 단위 : px, %, em, rem, vw, vh

width 의 단위

px : 그냥 픽셀, 한칸 한칸을 의미한다

% : 부모를 기준으로 한 %, 100% = 부모의 너비

em = 글씨 크기를 기준으로 한다, 부모의 글씨 크기

rem = 글씨 크기를 기준으로 한다, html의 글씨 크기, 모바일, PC를 한번에 만들때 많이 사용한다

ex) 글씨 크기가 16px, 그럼 1em, 1rem은 16px 글씨 크기가 32px, 1rem = 1em = 32px, 0.5rem = 16px

vw, vh : viewport, 브라우저의 크기나 스마트폰의 크기를 기준으로 % 100vw = 화면의 너비, 100vh = 화면의 높이

width : 영역의 너비
max-width : 영역의 최대 너비
min-width : 영역의 최소 너비
%, vw, vh은 기준이 있고 그 기준의 비율로 너비를 설정 한다.
어떤 내용을 넣었는데, 최소 100px은 있어야한다면 ? min-width를 사용
어떤 내용을 넣었는데, 너무 크면 안된다면 ? 즉, 1000px를 넘기면 안된다면 : max-width를 사용

height : 영역의 높이
max-height : 영역의 최대 높이
min-height : 영역의 최소 높이
이미지를 조절 할 때, 이미지에 대한 style을 할 때 다시 언급 할 예정

width 설정 값이 없다 ? 부모의 너비를 받아오는데 사용자가 margin 설정값을 10px로 설정했으니
margin(외부영역)을 우선적으로 만들어줘야하기에 부모의 너비가 아닌 설정값으로 적용 된다.

너비는 부모의 너비를 따라 받는다. 높이는 자식의 높이를 따라 간다. 즉, 설정을 하지 않으면 0px로 설정되기에 조심해야한다.
