현재 yarn 3.3버전이 상당히 불안정한 것 같다
create-react-app 이걸 쓰라고 한다.

npm list -g << 전역으로 설치된 라이브러리를 확인한다.

npm uninstall -g corepack << 전역으로 설치된 corepack을 삭제한다. yarn 3.3ver임

yarn create reacte-app testing << testing이라는 이름의 react 기본 설치
yarn crate react-app . << 현재 폴더에 react 기본 설치

yarn에서는 i가 아닌 add를 쓴다.

npm i styled-components === yarn add styled-components

npm remove styled-components === yarn remove styled-components 

npm install === yarn install
